Sobre

Este site é uma plataforma de chamada de ambulância para ajudar os pacientes a se comunicarem com o sistema de saúde local.

Como Funciona

  1. Clique no botão "Chamar Ambulância" para iniciar uma chamada.
  2. Fale com o atendente sobre sua situação e os detalhes da emergência.
  3. A ambulância será encaminhada imediatamente.
